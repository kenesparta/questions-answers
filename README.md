# 2. Environment file (.env)

1. Copy the .env-example file to .env file:

```shell
cp .env-example .env
```

2. Fill out the variables with your own credentials
```
API_PORT=8087

POSTGRES_DB=mydb
POSTGRES_USER=user
POSTGRES_PASSWORD=secret
POSTGRES_DRIVER=postgres
POSTGRES_HOST=127.0.0.1
```
